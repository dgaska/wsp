Site Intent

The intent of the redesign for this site is to make the site have an overall better presentation. Right now the site works but is missing some visual aesthetics. The goal is to make this website six pages with all the information the user is looking for, but also easy for the user to use. St. Francis Animal Hospital also would like the site more visually appealing to the user than it is now. The users of the site will want to be able to go to the site and find the information they need about the hospital. They the user would like to know what services the animal hospital provides, as well as the staff information so they know whom they are dealing with. If the user is comfortable using the site and have knowledge of whom and what the hospital offers the user will feel comfortable bringing their pet to St. Francis Hospital for Animals.

Demographics

The demographics for all of the sites tend to be the same. They are adults with pets looking for animal care. They are mainly living in the Charlotte, NC area. Based off of the data from the VCA Hospital there are more females visiting than males. The users have some college education and are usually browsing for home or work.

Target Audience

The target audience for this site is mainly residential users. The main target user is going to be adults, but from the demographic its more female than male. The user is going to be looking for an animal hospital that is cost efficient as well as some one they can trust. They will spending a short amount of time on the site so we want to make sure the audience can easily use the site to find what they want.
